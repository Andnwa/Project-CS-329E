# Project-CS-329E
My app is called BookBoard it is supposed to be a social media app that allows you to post a link that generates a QR Code to a playlist with the book title, user's name, username, and the author's name. 
I am using Version 12.4 of XCode on the iPhone 12 Pro Max my app does not supprt any other version it is strictly meant to be used in potrait mode. I am using Firestore, FireAuth, and CoreImage.

"Settings” screen. The two behaviors implemented are: switch that changes to dark mode and a slider that changes the font of text.
Some Non-default fonts and colors used
Two major elements used: Login/register path with Firebase and User Profile path using camera and photo library
Minor Elements used: sliders and switch
